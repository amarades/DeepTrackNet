# Resume-Ready Project Description — CrowdSafe AI

## One-Line Summary
> Built a production-grade AI crowd monitoring platform using YOLOv8, DeepSORT, FastAPI, and React with real-time predictive safety analytics and anomaly detection.

---

## Resume Bullet Points (Pick 3–5)

- Architected a **full-stack AI safety platform** detecting and tracking 200+ people/frame in real-time using YOLOv8 + DeepSORT, achieving 94%+ detection accuracy across 4 concurrent CCTV streams
- Engineered a **time-series forecasting system** using Prophet + LSTM ensemble predicting crowd density 30 minutes ahead with RMSE < 9, reducing incident response time by enabling proactive intervention
- Implemented an **SHAP-explainable anomaly detection module** using Isolation Forest to identify panic, surge, and reverse-flow events with AUC-ROC of 0.92 on held-out test data
- Built a **real-time React dashboard** with WebSocket streaming, canvas-rendered heatmaps, zone density analytics, and a multi-channel alert system (Email/SMS/Telegram)
- Designed a **scalable microservices architecture** using FastAPI, PostgreSQL, Redis, Docker Compose, and GitHub Actions CI/CD with deployment to AWS EC2 via ECR

## LinkedIn Project Description

**CrowdSafe AI — Crowd Monitoring & Predictive Safety Analytics Platform**

An end-to-end AI-powered public safety platform for real-time crowd monitoring and incident prevention. Designed for smart cities, stadiums, airports, and festival venues.

**Tech Stack:** Python · FastAPI · YOLOv8 · DeepSORT · Prophet · LSTM · React · Tailwind CSS · PostgreSQL · Redis · Docker · AWS

**Key Achievements:**
✅ Real-time detection of 200+ people per frame at 10 FPS
✅ Zone-based crowd density heatmaps with green/yellow/red risk visualization
✅ 30-minute crowd density and stampede risk forecasting
✅ AI-explainable alerts using SHAP feature attribution
✅ 4-role RBAC JWT authentication system
✅ Full Docker + CI/CD production deployment

## Hackathon / Competition Pitch (60 seconds)

Every year, crowd disasters kill hundreds of people — Astroworld, Itaewon, Kanjuruhan.
Current systems react after incidents begin. CrowdSafe AI *predicts* them 30 minutes before.

Using YOLOv8 for detection, DeepSORT for tracking, and a Prophet+LSTM ensemble for forecasting — our platform gives security teams an early warning system with AI-explained alerts and actionable recommendations.

Built with FastAPI + React, deployable via Docker in under 5 minutes.
